HTML5 & CSS3 & JS (ES6+)

HTML5 ile semantik ve erişilebilir web yapıları oluşturabiliyorum. CSS3 tarafında Flexbox ve Grid kullanarak modern, responsive arayüzler geliştirebiliyorum.

JavaScript (ES6+) ile dinamik ve etkileşimli kullanıcı arayüzleri geliştirebiliyor, modüler ve temiz kod yapısına önem veriyorum. DOM manipülasyonu, async/await yapısı ve API entegrasyonları ile modern web uygulamaları oluşturabiliyorum.

UI tasarımında özellikle kullanıcı deneyimi (UX) ve mobil uyumluluğa önem veriyorum.

html · MySQL · PostgreSQL · WordPress

html ile backend tarafında dinamik ve veri odaklı web uygulamaları geliştirebiliyorum. Form işlemleri, oturum yönetimi (session), kullanıcı doğrulama ve sunucu taraflı işlemler konusunda aktif olarak çalışabiliyorum.

MySQL ve PostgreSQL ile ilişkisel veritabanı tasarımı yapabiliyor, veri normalizasyonu, sorgu optimizasyonu ve CRUD işlemlerini etkin bir şekilde yönetebiliyorum.

WordPress ile tema özelleştirme ve temel eklenti yapılarını kullanarak içerik yönetim sistemleri (CMS) tabanlı web siteleri geliştirebiliyorum.

React & Laravel Frameworks

React ile bileşen tabanlı modern kullanıcı arayüzleri geliştirebiliyor, state management yapıları ve hook'lar sayesinde dinamik ve ölçeklenebilir frontend uygulamaları oluşturabiliyorum.

Laravel ile MVC mimarisine uygun, güvenli ve sürdürülebilir backend uygulamaları geliştirebiliyorum. Routing, middleware, Eloquent ORM ve authentication gibi yapılarını etkin şekilde kullanabiliyorum.

Git & GitHub

Git ile versiyon kontrol sistemi kullanarak projelerimde yapılan değişiklikleri düzenli bir şekilde takip edebiliyor, dallanma (branch) yapısı sayesinde farklı geliştirme süreçlerini güvenli şekilde yönetebiliyorum.

GitHub ile projelerimi merkezi bir platformda saklayarak ekip çalışmasına uygun hale getirebiliyor, repository yönetimi, pull request ve issue takibi gibi araçları etkin şekilde kullanabiliyorum.
